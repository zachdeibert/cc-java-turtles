Turtle Start Guide

Place your turtle, then type in the following: 

> edit download
server = "127.0.0.1:{{ port }}"
local out = fs.open("startup", "w")
local in  = http.get("http://" .. server .. "/lua")
out.write(in.readAll())
in.close()
out.close()

Then, press ctrl, then enter, then ctrl, left arrow, enter to save and exit the editor. Finally, type in the following: 

> download
> reboot
